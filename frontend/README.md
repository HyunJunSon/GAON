# GAON Frontend

## Runtime & Package Manager

- Node: 20.11.1 (`.nvmrc`, `.node-version`)
- pnpm: 9.12.1 (`packageManager` + Corepack)

## Setup

```bash
nvm use
corepack enable
pnpm -v
pnpm i
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
